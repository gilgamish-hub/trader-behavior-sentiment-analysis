# Trader Behavior Analysis using Bitcoin Fear & Greed Index

This project analyzes the relationship between cryptocurrency trader behavior and market sentiment.

## Key Insights
• Traders are more active during Greed sentiment  
• Average profits are higher during Greed markets  
• Short trades perform better during Fear markets  
• A small number of traders generate most profits

## Tools Used

Pandas
Seaborn
